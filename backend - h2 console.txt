H2 database: http://localhost:8080/h2-console

Start: POST http://localhost:8080/api/start

Check status: GET http://localhost:8080/api/status

Stop: POST http://localhost:8080/api/stop

---------------------------------------------

Endpoints:
POST /api/start: Starts the ticketing system by initiating vendor and customer threads.
POST /api/stop: Stops the ticketing system by terminating all running threads.
GET /api/status: Retrieves the current status of the ticketing system.
